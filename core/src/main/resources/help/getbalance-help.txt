getbalance

NAME
----
getbalance - get wallet balance(s)

SYNOPSIS
--------
getbalance
		[--currency-code=<bsq|btc>]

DESCRIPTION
-----------
Returns full balance information for KOINZ KNZ and/or BTC wallets.

OPTIONS
-------
--currency-code=<bsq|btc>
		The three letter KOINZ wallet crypto currency code.

EXAMPLES
--------
Show full KNZ and BTC wallet balance information:
$ ./koinz-cli --password=xyz --port=9998 getbalance

Show full KNZ wallet balance information:
$ ./koinz-cli --password=xyz --port=9998 getbalance --currency-code=bsq

Show full BTC wallet balance information:
$ ./koinz-cli --password=xyz --port=9998 getbalance --currency-code=btc
