removewalletpassword

NAME
----
removewalletpassword - remove a KOINZ wallet's encryption password

SYNOPSIS
--------
removewalletpassword
        --wallet-password=<wallet-password>
        --timeout=<seconds>

DESCRIPTION
-----------
Remove an encryption password from an encrypted KOINZ wallet.

EXAMPLES
--------
$ ./koinz-cli --password=xyz --port=9998 removewalletpassword --wallet-password=mypassword
