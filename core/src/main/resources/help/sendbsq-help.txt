sendbsq

NAME
----
sendbsq - send KNZ to an external wallet

SYNOPSIS
--------
sendbsq
		--address=<bsq-address>
		--amount=<bsq-amount>
		[--tx-fee-rate=<sats/byte>]

DESCRIPTION
-----------
Send KNZ from your KOINZ wallet to an external KNZ address.

OPTIONS
-------
--address
        The destination KNZ address for the send transaction.

--amount
		The amount of KNZ to send.

--tx-fee-rate
        An optional transaction fee rate (sats/byte) for the transaction.  The user is
        responsible for choosing a fee rate that will be accepted by the network in a
        reasonable amount of time, and the fee rate must be greater than 1 (sats/byte).

EXAMPLES
--------
Send 500 KNZ to address Bn3PCQgRwhkrGnaMp1RYwt9tFwL51YELqne with a default transaction fee rate:
$ ./koinz-cli --password=xyz --port=9998 sendbsq --address=Bn3PCQgRwhkrGnaMp1RYwt9tFwL51YELqne --amount=500.00

Send 3000 KNZ to address Bn3PCQgRwhkrGnaMp1RYwt9tFwL51YELqne with transaction fee rate of 40 sats/byte:
$ ./koinz-cli --password=xyz --port=9998 sendbsq --address=Bn3PCQgRwhkrGnaMp1RYwt9tFwL51YELqne --amount=3000.00 \
    --tx-fee-rate=40
