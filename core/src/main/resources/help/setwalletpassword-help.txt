setwalletpassword

NAME
----
setwalletpassword - set KOINZ wallet password

SYNOPSIS
--------
setwalletpassword
        --wallet-password=<wallet-password>
        --new-wallet-password=<new-wallet-password>

DESCRIPTION
-----------
Encrypts a KOINZ wallet with a password.  If the optional new wallet password option is
present, a new wallet password replaces the existing password

EXAMPLES
--------
Encrypt an unencrypted KOINZ wallet with a password:
$ ./koinz-cli --password=xyz --port=9998 setwalletpassword --wallet-password=mypassword

Set a new password on a KOINZ wallet that is already encrypted:
$ ./koinz-cli --password=xyz --port=9998 setwalletpassword --wallet-password=myoldpassword \
    --new-wallet-password=mynewpassword
